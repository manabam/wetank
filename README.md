### 하이퍼링크
```html
<a href="http://www.naver.com" target="_blank" title="네이버 홈페이지 열기">네이버</a>
```
target="_blank"   ---> 새 페이지에서 네이버가 띄워짐
title             ---> 마우스를 가져다 대면 나오는 글
